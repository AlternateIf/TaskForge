# TaskForge Keyboard Shortcuts

## Global Shortcuts (Work Anywhere)
| Key | Action |
|-----|--------|
| `?` | Show shortcut reference overlay |
| `/` | Focus search bar |
| `g d` | Go to dashboard |
| `g p` | Go to projects list |
| `n` | New task (opens create dialog) |
| `Escape` | Close modal/panel/overlay |

## Board/List View Shortcuts
| Key | Action |
|-----|--------|
| `j` | Navigate up in task list |
| `k` | Navigate down in task list |
| `Enter` | Open selected task detail |
| `1` | Set priority to Critical |
| `2` | Set priority to High |
| `3` | Set priority to Medium |
| `4` | Set priority to Low |
| `5` | Set priority to None |

## Task Detail Shortcuts
| Key | Action |
|-----|--------|
| `a` | Assign task (opens assignee picker) |
| `l` | Add label (opens label picker) |
| `s` | Change status (opens status picker) |
| `c` | Focus comment input |
| `e` | Edit description |

## Command Palette
| Key | Action |
|-----|--------|
| `Ctrl+K` / `Cmd+K` | Open command palette |

## Notes
- Shortcuts are disabled when typing in input fields (except Escape)
- Press `g` then `d` within 500ms to navigate to dashboard
- Press `g` then `p` within 500ms to navigate to projects
- Shortcuts don't interfere with browser or screen reader hotkeys
