# TaskForge — Style Guide Router

Use this router to avoid loading the full visual corpus for routine frontend tasks.

## Read Order
1. `styleguide-core.md` for daily implementation (tokens, component behavior, accessibility, responsive rules).
2. `styleguide-extended.md` only when work explicitly involves advanced visual language, branding assets, icon maps, or view-specific art direction.

## When To Load Extended
Load `styleguide-extended.md` only for:
- logo/iconography decisions
- illustration/empty-state direction
- highly bespoke layout styling and premium visual polish work
- view-specific visual choreography not covered by core primitives

## Source of Truth
- Core = implementation defaults.
- Extended = deeper design language and brand system details.
